O que é HTML?

O HTML (HyperText Markup Language) é a linguagem utilizada para estruturar páginas da web. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação.

Utilizamos tags para organizar o conteúdo da página, como títulos, parágrafos, imagens e links.

Exemplo de formatação de texto

No HTML podemos destacar textos utilizando algumas tags:

Texto em negrito (strong)
Texto em destaque ou itálico (em)

Essas tags ajudam a dar significado ao conteúdo da página.

Quebra de linha

Para pular linhas dentro de um parágrafo usamos a tag <br>.

Exemplo:
Linha 1
Linha 2
Linha 3

Links na Web

Um dos elementos mais importantes da web são os links. Eles permitem navegar entre páginas diferentes.

Exemplo de link que faz uma busca no Google:

Buscar "Minecraft" no Google

Buscar "Jogo Preferido" no Google

Link para outra página do site

Links relativos

Links relativos são usados para navegar entre arquivos dentro do próprio projeto.

Exemplos:

Página inicial na mesma pasta: Index

Arquivo dentro de uma pasta: Dica 1

Voltar uma pasta: Voltar
