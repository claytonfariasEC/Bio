O que é HTML?

O HTML (HyperText Markup Language) é a linguagem utilizada para estruturar páginas da web. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação.

Dicas para aprender HTML

Algumas dicas importantes para quem está começando:

✔ Pratique criando páginas simples
✔ Teste o código diretamente no navegador
✔ Leia e entenda as tags HTML
✔ Cometa erros e tente corrigir (isso ajuda muito a aprender)
